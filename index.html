HTTP/HTTPS basic auth proxy

This serivce provides authentication using GET parameter instead of HTTP/HTTPS basic auth.

Source code

https://github.com/matsubo/http-basic-auth-proxy/

Usage

Endpoint URL

https://http-basic-auth-proxy.herokuapp.com/proxy.php

Parameters

Key Description Example
http_basic_user (required) HTTP Basic auth user name username
http_basic_password (required) HTTP Basic auth password passw0rd
http_host_path (required) destination of host name and path example.com/actual/endpoint
http_scheme (optional) Protocol of destination endpoint. default is https. https

All GET parameters are passed to destination URL.

Example

https://http-basic-auth-proxy.herokuapp.com/proxy.php?http_basic_user=username&http_basic_password=passw0rd&http_host_path=example.com/&additional=parameters&
